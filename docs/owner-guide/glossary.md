# Technical & Business Glossary

This glossary explains technical terms in simple language for non-technical stakeholders.

---

| Term | Simple Explanation |
| :--- | :--- |
| **App Router** | Next.js navigation system that organizes web pages into folders and delivers fast page loads. |
| **Authentication** | The process of verifying who a user is (e.g. logging in with email/password or Google). |
| **Authorization** | The security rules that determine what a logged-in user can do (e.g. customer vs. store admin). |
| **CMP (Consent Management Platform)** | The privacy system that displays the cookie banner, records customer choices, and blocks tracking scripts until permission is granted. |
| **CDP (Customer Data Platform)** | The analytics system that records customer events, search queries, and shopping funnels to generate business reports. |
| **Database** | The secure storage engine where all products, orders, customers, and inventory data are saved. |
| **Drizzle ORM** | A software tool that allows XINVORA's code to safely talk to the database without syntax errors. |
| **HMAC Signature** | A digital security seal placed on cookies to prevent customers or hackers from tampering with their browser data. |
| **IP Hashing** | A privacy technique that converts customer IP addresses into random cryptographic strings (`SHA-256`) to comply with GDPR privacy laws. |
| **Server Action** | A secure function that runs directly on the server to process form submissions (like adding to cart or updating settings). |
| **SKU (Stock Keeping Unit)** | A unique identification code assigned to a specific product variant (e.g. `XIN-DRS-BLK-M`). |
| **Zod Validation** | A data checker that verifies all incoming information (like names, emails, prices) before it touches the database. |
